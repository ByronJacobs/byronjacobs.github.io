Numerical Analysis B

Chapter 2 Taylor Series (Revisted)

Taylor’s theorem and the Taylor series of a function provide an analysis framework for a broad class of numerical methods. Primarily “finite difference” methods.

Theorem 2.1 Taylor’s thoerem for functions of one1 real variable Assume that \(f(x)\) has \(n+1\) continuous derivatives for \(a \le x \le b\), and let \(x_0 \in [a,b]\) then \[ f(x) = p_n(x) + R_n(x), \quad a \le x \le b, \] where \[ p_n(x) = f(x_0) + (x-x_0) f'(x_0) + \frac{(x-x_0)^2}{2!}f''(x_0) + \ldots + \frac{(x-x_0)^n}{n!} f^{(n)}(x_0), \] is the Taylor polynomial of degree \(n\) for the function \(f(x)\) and the point of approximation \(x_0\), and \(R_n(x)\) is the remainder in approximating \(f(x)\) by \(p_n(x)\). We have \[\begin{align} R_n(x) &= \frac{1}{n!} \int_{x_0}^{x} (x-\xi)^nf^{(n+1)}(\xi) d\xi \\ &= \frac{(x-x_0)^{n+1}}{(n+1)!} f^{(n+1)}(c_x), \end{align}\] where \(c_x\) is an unknown point between \(x_0\) and \(x\).

This result may be interpreted in a number of ways. The first illustrates that, if a function (and all necessary derivatives) are known at a point then the function can be reconstructed anywhere in the domain. Consider,

\[\begin{equation} f(x) = f(x_0) + (x-x_0)f'(x_0) + \frac{(x-x_0)^2}{2!} f''(x_0) + \mathcal{O}{(x-x_0)^3}. \end{equation}\]

if \(f(x_0), f'(x_0), \ldots\) are given \(f(x)\) can be found for any value of \(x\). Obviously, in practice, we can’t know arbitrarily many function (and derivative) values. As such, we will incur an error in our approximation which is proportional to the distance between \(x\) and \(x_0\), shown explicitly by the \(\mathcal{O}\) term.

Example 2.1 Consider \(f(x) = e^x\), then \(f^{(n)}(x) = e^x \quad \forall n\). Find the series approximation to \(f(0.25)\) using four terms of Taylor’s series, recalling that \[ e^x = \sum_{n=0}^\infty \frac{x^n}{n!}. \]

Solution: Take \(x=0\) and \(\Delta x=0.25\), then from Taylor’s series we have \[\begin{align} f(x+\Delta x) &\approx f(x) + \Delta x f'(x) + \frac{\Delta x^2}{2!} f''(x) + \frac{\Delta x^3}{3!} f'''(x) \\ f(0.25) &\approx f(0) + 0.25 f'(0) + \frac{0.25^2}{2!} f''(0) + \frac{0.25^3}{3!} f'''(0) \\ &= 1.28385 \end{align}\] Where the exact value is \(e^{0.25} = 1.28403\). 
import numpy as np
def taylorExp(n,dx):
  ns = np.arange(n)
  num = (dx)**(ns)
  den = np.array([ np.math.factorial(n) for n in ns])
  return np.sum(num/den)

n = 4
dx = 0.25
res = taylorExp(n,dx)
print(f"for n = {n} and dx = {dx} we get, {res}.")
## for n = 4 and dx = 0.25 we get, 1.2838541666666667.

The second interpretation of Taylor’s series is allowing one to define the relationship between a function’s derivatives and discrete sampling of the function. This will form the basis of constructing finite difference methods, by allowing us to discretise differential operators. With appropriate substitution we have \[\begin{equation} f(x+\Delta x) = f(x) + \Delta x f'(x) + \frac{\Delta x^2}{2!} f''(x) + \ldots + \mathcal{O}(\Delta x^n), \end{equation}\] rearranging we find, \[\begin{equation} f'(x) = \frac{f(x+\Delta x) - f(x)}{\Delta x} + \mathcal{O}(\Delta x). \end{equation}\]
